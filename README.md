# SPZ — Системне Програмне Забезпечення для Проактивного Аналізу Стану Windows-системи

> Бакалаврська робота | C++ | Qt 6 | Win32 API | PDH

---

## Про проект

Цей проект є **системним монітором** (аналогом **Process Explorer**) для ОС Windows, розробленим на C++ з використанням фреймворку **Qt 6**.

Програма дозволяє в реальному часі відстежувати:
- Список активних процесів з відображенням їх споживання CPU та RAM
- Системне навантаження (CPU, RAM, GPU) у вигляді живих графіків
- Системні події: підключення USB-пристроїв, зміни живлення
- Мережеві події: зміна підключення до Інтернету
- Файлові події: зміни у директорії `C:\Users`
- Запуск та завершення процесів

## Архітектура проекту

```
BKR/
├── SPZ_Qt/                  ← Основний проект (Qt6 версія)
│   ├── CMakeLists.txt
│   ├── CMakePresets.json
│   └── src/
│       ├── main.cpp         ← Точка входу, COM ініціалізація, DeviceNotification
│       ├── backend.h/.cpp   ← Моніторинг (процеси, CPU, RAM, GPU, USB, мережа, файли)
│       ├── mainwindow.h/.cpp← GUI: вкладки, графіки, таблиці
│       └── process_info.h   ← Структури даних (ProcessData, SystemData)
│
└── SPZ_Merged/              ← Попередня Win32 API версія (резервна копія)
```

### Ключові патерни

- **Backend** — клас (`QObject`), що запускає таймери та потоки для збору даних і надсилає їх через **Qt Signals & Slots**  
- **MainWindow** — підключається до сигналів Backend і безпечно оновлює GUI з головного потоку  
- **FsWorker** — окремий Qt-потік для блокуючого очікування файлових змін (`FindFirstChangeNotification`)

## Як запустити

### Передумови

1. **Qt 6.x** — встановіть через [Qt Online Installer](https://www.qt.io/download-qt-installer), компонент `MSVC 2022 64-bit`
2. **Visual Studio 2022** з встановленим модулем *Desktop development with C++*
3. **CMake** (зазвичай входить до складу Qt або VS)

### Кроки

```bash
# 1. Клонуйте репозиторій
git clone https://github.com/Alpachinka/BKR_System-Software-for-Proactive-Windows-System-State-Analysis.git
cd BKR_System-Software-for-Proactive-Windows-System-State-Analysis

# 2. Відкрийте папку SPZ_Qt у Visual Studio 2022
#    File → Open → Folder... → оберіть SPZ_Qt/

# 3. Visual Studio знайде CMakeLists.txt і автоматично налаштує проект
#    (CMakePresets.json вже вказує шлях C:/Qt/6.11.0/msvc2022_64)

# 4. Натисніть кнопку ▶ SPZ_Qt.exe або F5

# 5. Після першого запуску виконайте windeployqt (копіює Qt DLL):
C:\Qt\6.11.0\msvc2022_64\bin\windeployqt.exe out\build\Debug\SPZ_Qt.exe
```

## Функціонал

### Вкладка "Процеси (Активні)"
- Список усіх запущених процесів
- Відображення споживання RAM (МБ) та CPU (%)
- Сортування за будь-якою колонкою

### Вкладка "Системні Ресурси"
- Таблиця з даними про RAM та диск C:
- Три кольорові шкали (CPU/RAM/GPU)
- Єдиний живий графік CPU + RAM + GPU за останні 60 секунд (синій / зелений / помаранчевий)

### Вкладка "Логи" (4 підвкладки)
| Підвкладка | Що відстежується |
|---|---|
| 📋 Процеси | Запуск та завершення програм |
| 🖥 Система | USB підключення / відключення, зміна живлення |
| 🌐 Мережа | Підключення / відключення від Інтернету |
| 📁 Файли | Зміни у файлах `C:\Users` |

Усі логи можна зберегти у CSV через кнопку **"Зберегти"**.
